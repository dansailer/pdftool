## About Priska PDF Tool

Priska PDF Tool is a sleek, cross-platform PDF manipulation application inspired by the intuitive interface of Mac Preview. It offers a simple yet powerful way to handle PDF documents on macOS, Windows, and Linux.

---

## Version

**Current Version:** 0.9.2 (Release Candidate)

All core features are complete and the application is undergoing cross-platform testing.

---

## Features at a Glance

- **View PDFs** with high-quality rendering
- **Merge** multiple PDF files into one
- **Reorder** pages with drag-and-drop
- **Delete** unwanted pages
- **Rotate** pages in any direction
- **Undo/Redo** all operations
- **Dark/Light theme** support
- **Document metadata** editing
- **Auto-updates** from GitHub Releases

## Technology

Priska PDF Tool is built with modern, cross-platform technologies:

| Component | Technology |
|-----------|------------|
| Application Framework | [Tauri](https://tauri.app) 2.x |
| Backend Language | [Rust](https://www.rust-lang.org) |
| Frontend Language | TypeScript |
| PDF Rendering | [pdf.js](https://mozilla.github.io/pdf.js/) |
| PDF Manipulation | [lopdf](https://github.com/J-F-Liu/lopdf) |

## Privacy

**Your data stays private.** Priska PDF Tool:

- Works completely offline
- Never uploads your files to any server
- Processes all PDFs locally on your computer
- Only connects to the internet for update checks (to GitHub)
