Priska PDF Tool supports comprehensive keyboard shortcuts for efficient editing. Learn these shortcuts to speed up your workflow.

---

## File Operations {#file-operations}

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Open file(s) | `Cmd + O` | `Ctrl + O` |
| Save As | `Cmd + S` | `Ctrl + S` |
| Close window | `Cmd + W` | `Ctrl + W` |

---

## Editing {#editing}

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Undo | `Cmd + Z` | `Ctrl + Z` |
| Redo | `Cmd + Shift + Z` | `Ctrl + Shift + Z` |
| Delete selected page(s) | `Delete` or `Backspace` | `Delete` or `Backspace` |
| Select all pages | `Cmd + A` | `Ctrl + A` |

## Tips for Efficient Editing

### Batch Operations
1. Select multiple pages using Shift+Click or Cmd/Ctrl+Click
2. Perform operations (delete, rotate) on all selected pages at once

### Quick Reordering
Use Shift+Arrow keys to move selected pages up or down without using the mouse.

### Undo Freely
Don't be afraid to experiment! With 50 levels of undo, you can always go back.

### Keyboard-Only Workflow
You can navigate and edit entirely with the keyboard:
1. Use arrow keys to navigate between pages
2. Use Shift+Click or Cmd/Ctrl+A to select
3. Use Delete to remove pages
4. Use Shift+Arrows to reorder
5. Use Cmd/Ctrl+Z to undo mistakes
6. Use Cmd/Ctrl+S to save
