## Installation

### macOS

1. Laden Sie die `.dmg`-Datei von der [Releases-Seite](https://github.com/dansailer/pdftool/releases) herunter
2. Oeffnen Sie die heruntergeladene DMG-Datei
3. Ziehen Sie **Priska PDF Tool** in Ihren Programme-Ordner
4. Beim ersten Start: Rechtsklicken Sie auf die App und waehlen Sie "Oeffnen", um Gatekeeper zu umgehen

**Hinweis:** Die App ist ein Universal Binary, das nativ auf Intel- und Apple-Silicon-Macs laeuft.

### Windows

1. Laden Sie das `.msi`-Installationsprogramm von der [Releases-Seite](https://github.com/dansailer/pdftool/releases) herunter
2. Fuehren Sie das Installationsprogramm aus und folgen Sie den Anweisungen
3. Starten Sie Priska PDF Tool aus dem Startmenue

### Linux

**AppImage:**
1. Laden Sie die `.AppImage`-Datei von der [Releases-Seite](https://github.com/dansailer/pdftool/releases) herunter
2. Machen Sie sie ausfuehrbar: `chmod +x Priska-PDF-Tool.AppImage`
3. Fuehren Sie das AppImage aus

**Debian/Ubuntu:**
1. Laden Sie das `.deb`-Paket von der [Releases-Seite](https://github.com/dansailer/pdftool/releases) herunter
2. Installieren Sie mit: `sudo dpkg -i priska-pdf-tool_*.deb`

## Dateien oeffnen {#opening-files}

Es gibt mehrere Moeglichkeiten, PDF-Dateien in Priska PDF Tool zu oeffnen:

### Drag and Drop
Ziehen Sie einfach eine oder mehrere PDF-Dateien aus Ihrem Dateimanager und legen Sie sie auf dem Anwendungsfenster ab. Mehrere Dateien werden automatisch in der Reihenfolge zusammengefuehrt, in der sie abgelegt wurden.

### Datei-Menue
1. Klicken Sie auf **Datei** in der Menueleiste
2. Waehlen Sie **Oeffnen...** (oder druecken Sie `Cmd+O` / `Strg+O`)
3. Waehlen Sie eine oder mehrere PDF-Dateien im Dateidialog aus
4. Klicken Sie auf **Oeffnen**

### Mehrere Dateien oeffnen
Wenn Sie mehrere PDF-Dateien oeffnen, werden diese automatisch zu einem einzigen Dokument kombiniert. Sie koennen dann:
- Seiten durch Ziehen der Miniaturansichten neu anordnen
- Unerwuenschte Seiten loeschen
- Das kombinierte Ergebnis als neue PDF speichern

## Grundlegende Navigation {#basic-navigation}

### Seiten anzeigen

Das Hauptfenster besteht aus zwei Bereichen:

- **Miniaturansicht-Panel (Links):** Zeigt kleine Vorschauen aller Seiten. Klicken Sie auf eine Miniaturansicht, um zu dieser Seite zu navigieren.
- **Hauptanzeige (Rechts):** Zeigt die aktuell ausgewaehlte Seite in voller Groesse an.

### Zwischen Seiten navigieren

| Methode | Beschreibung |
|---------|--------------|
| Miniaturansicht klicken | Direkt zu einer beliebigen Seite springen |
| Auf/Ab-Pfeiltasten | Zur vorherigen/naechsten Seite wechseln |
| Scrollen | Durch die Hauptanzeige scrollen |

### Zoom-Steuerung

Die Symbolleiste bietet Zoom-Steuerungen:

- **Vergroessern (+):** Vergroesserung erhoehen
- **Verkleinern (-):** Vergroesserung verringern
- **An Breite anpassen:** Seite an Fensterbreite anpassen

Sie koennen auch **Pinch-to-Zoom**-Gesten auf Trackpads verwenden.

### Mehrere Seiten auswaehlen

Um mehrere Seiten fuer Stapeloperationen auszuwaehlen:

- **Umschalt+Klick:** Einen Bereich von Seiten auswaehlen
- **Cmd+Klick / Strg+Klick:** Einzelne Seitenauswahl umschalten

Ausgewaehlte Seiten werden im Miniaturansicht-Panel hervorgehoben.

## Ihre erste Bearbeitung

Lassen Sie uns einen einfachen Bearbeitungsablauf durchgehen:

### 1. Ihre PDFs oeffnen
Ziehen Sie zwei oder mehr PDF-Dateien auf das Fenster. Sie werden automatisch zusammengefuehrt.

### 2. Seiten neu anordnen
Ziehen Sie Miniaturansichten im Miniaturansicht-Panel nach oben oder unten, um Seiten neu anzuordnen. Sie koennen auch **Umschalt+Auf/Ab**-Pfeiltasten verwenden.

### 3. Unerwuenschte Seiten loeschen
Waehlen Sie Seiten aus, die Sie entfernen moechten, und druecken Sie **Entf** oder **Ruecktaste**, oder klicken Sie auf die Loeschen-Schaltflaeche in der Symbolleiste.

### 4. Seiten drehen
Waehlen Sie Seiten aus und verwenden Sie die Drehen-Schaltflaechen in der Symbolleiste, um um 90 Grad nach links oder rechts zu drehen.

### 5. Ihre Arbeit speichern
1. Gehen Sie zu **Datei > Speichern unter...** (oder druecken Sie `Cmd+S` / `Strg+S`)
2. Fuellen Sie die optionalen Metadaten aus (Titel, Autor, Betreff, Schluesselwoerter)
3. Waehlen Sie einen Speicherort und Dateinamen
4. Klicken Sie auf **Speichern**
