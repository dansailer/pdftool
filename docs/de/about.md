## Ueber Priska PDF Tool

Priska PDF Tool ist eine elegante, plattformuebergreifende PDF-Bearbeitungsanwendung, inspiriert von der intuitiven Oberflaeche von Mac Preview. Es bietet eine einfache, aber leistungsstarke Moeglichkeit, PDF-Dokumente auf macOS, Windows und Linux zu bearbeiten.

---

## Version

**Aktuelle Version:** 0.9.2 (Release Candidate)

Alle Kernfunktionen sind abgeschlossen und die Anwendung wird plattformuebergreifend getestet.

---

## Funktionen im Ueberblick

- **PDFs anzeigen** mit hochwertiger Darstellung
- **Zusammenfuehren** mehrerer PDF-Dateien zu einer
- **Neuanordnen** von Seiten per Drag-and-Drop
- **Loeschen** unerwuenschter Seiten
- **Drehen** von Seiten in beliebige Richtung
- **Rueckgaengig/Wiederholen** aller Operationen
- **Dunkles/Helles Design**-Unterstuetzung
- **Dokument-Metadaten**-Bearbeitung
- **Automatische Updates** von GitHub Releases

---

## Technologie

Priska PDF Tool ist mit modernen, plattformuebergreifenden Technologien gebaut:

| Komponente | Technologie |
|------------|-------------|
| Anwendungs-Framework | [Tauri](https://tauri.app) 2.x |
| Backend-Sprache | [Rust](https://www.rust-lang.org) |
| Frontend-Sprache | TypeScript |
| PDF-Darstellung | [pdf.js](https://mozilla.github.io/pdf.js/) |
| PDF-Manipulation | [lopdf](https://github.com/J-F-Liu/lopdf) |

## Datenschutz

**Ihre Daten bleiben privat.** Priska PDF Tool:

- Funktioniert vollstaendig offline
- Laed Ihre Dateien niemals auf irgendeinen Server hoch
- Verarbeitet alle PDFs lokal auf Ihrem Computer
- Verbindet sich nur fuer Update-Pruefungen mit dem Internet (zu GitHub)
