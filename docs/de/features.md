Priska PDF Tool bietet umfassende Funktionen zum Anzeigen und Bearbeiten von PDF-Dokumenten. Hier ist alles, was Sie tun koennen:

---

## PDF-Anzeige {#viewing}

### Hochwertige Darstellung
Priska PDF Tool verwendet pdf.js fuer hochwertige PDF-Darstellung, die folgendes genau anzeigt:
- Text und Schriften
- Vektorgrafiken
- Bilder und Fotos
- Komplexe Layouts

### Miniaturansicht-Navigation
Das linke Panel zeigt Miniaturvorschauen aller Seiten in Ihrem Dokument, was es einfach macht:
- Einen Ueberblick ueber Ihr gesamtes Dokument zu erhalten
- Mit einem einzigen Klick direkt zu einer beliebigen Seite zu springen
- Die Auswirkungen Ihrer Bearbeitungen in Echtzeit zu sehen

### Zoom-Steuerung
Passen Sie die Ansicht nach Ihren Wuenschen an:
- **Vergroessern / Verkleinern:** Vergroesserung erhoehen oder verringern
- **An Breite anpassen:** Seite automatisch an Fensterbreite anpassen
- **Pinch-to-Zoom:** Verwenden Sie Trackpad-Gesten fuer praezise Zoom-Steuerung

## PDFs zusammenfuehren {#merging}

Kombinieren Sie mehrere PDF-Dokumente zu einer einzigen Datei:

### So fuehren Sie zusammen
1. **Drag and Drop** mehrerer PDF-Dateien auf das Fenster, ODER
2. Verwenden Sie **Datei > Oeffnen**, um mehrere Dateien gleichzeitig auszuwaehlen

Die PDFs werden in der Reihenfolge kombiniert, in der sie hinzugefuegt wurden. Sie koennen die Seiten dann nach Bedarf neu anordnen.

### Tipps zum Zusammenfuehren
- Es gibt keine Begrenzung fuer die Anzahl der PDFs, die Sie zusammenfuehren koennen
- Grosse Dateien benoetigen moeglicherweise etwas mehr Zeit zur Verarbeitung
- Das Miniaturansicht-Panel zeigt alle Seiten aus allen zusammengefuehrten Dokumenten
- Verwenden Sie Drag-and-Drop im Miniaturansicht-Panel, um Seiten aus verschiedenen Quelldateien neu anzuordnen

## Seiten neu anordnen {#reordering}

Ordnen Sie Seiten in Ihrem Dokument mit intuitivem Drag-and-Drop neu an:

### Mit der Maus
1. Klicken Sie auf eine Miniaturansicht, um sie auszuwaehlen
2. Ziehen Sie die Miniaturansicht an ihre neue Position
3. Ein visueller Indikator zeigt, wo die Seite eingefuegt wird
4. Lassen Sie los, um die Seite abzulegen

### Mit der Tastatur
1. Waehlen Sie die Seite(n) aus, die Sie verschieben moechten
2. Druecken Sie **Umschalt+Pfeil nach oben**, um nach oben zu verschieben
3. Druecken Sie **Umschalt+Pfeil nach unten**, um nach unten zu verschieben

### Mehrere Seiten verschieben
1. Waehlen Sie mehrere Seiten mit **Umschalt+Klick** (Bereich) oder **Cmd/Strg+Klick** (umschalten)
2. Ziehen Sie die Auswahl an die neue Position
3. Alle ausgewaehlten Seiten werden zusammen verschoben und behalten ihre relative Reihenfolge bei

## Seiten loeschen {#deleting}

Entfernen Sie unerwuenschte Seiten aus Ihrem Dokument:

### So loeschen Sie
1. Waehlen Sie die Seite(n) aus, die Sie loeschen moechten
2. Entweder:
   - Druecken Sie die **Entf**- oder **Ruecktaste**
   - Klicken Sie auf die **Loeschen**-Schaltflaeche in der Symbolleiste

### Mehrere Seiten loeschen
Waehlen Sie zuerst mehrere Seiten aus und loeschen Sie sie dann alle auf einmal. Dies ist nuetzlich fuer:
- Entfernen leerer Seiten
- Extrahieren nur der benoetigten Seiten
- Entfernen von Deckblaettern oder Anhaengen

### Loeschung rueckgaengig machen
Versehentlich eine Seite geloescht? Druecken Sie **Cmd+Z** / **Strg+Z**, um sofort rueckgaengig zu machen.

---

## Seiten drehen {#rotating}

Drehen Sie einzelne Seiten oder Auswahlen:

### So drehen Sie
1. Waehlen Sie die Seite(n) aus, die Sie drehen moechten
2. Klicken Sie auf die Drehen-Schaltflaechen in der Symbolleiste:
   - **Nach links drehen:** 90° gegen den Uhrzeigersinn
   - **Nach rechts drehen:** 90° im Uhrzeigersinn

### Drehverhalten
- Die Drehung erfolgt in 90-Grad-Schritten
- Mehrere Drehungen summieren sich (z.B. zwei Rechtsdrehungen = 180°)
- Die Drehung wird beim Speichern der PDF beibehalten

## Rueckgaengig/Wiederholen {#undo-redo}

Vollstaendige Rueckgaengig- und Wiederholen-Unterstuetzung fuer alle Bearbeitungsvorgaenge:

### Unterstuetzte Vorgaenge
- Seiten loeschen
- Seiten neu anordnen (einzeln und mehrere)
- Seiten drehen

### Verwendung
- **Rueckgaengig:** Druecken Sie **Cmd+Z** / **Strg+Z** oder verwenden Sie **Bearbeiten > Rueckgaengig**
- **Wiederholen:** Druecken Sie **Cmd+Umschalt+Z** / **Strg+Umschalt+Z** oder verwenden Sie **Bearbeiten > Wiederholen**

### Verlaufslimit
Der Rueckgaengig-Verlauf speichert bis zu **50 Aktionen**. Aeltere Aktionen werden automatisch entfernt, um Speicher zu sparen.

## Dunkles/Helles Design {#themes}

Waehlen Sie Ihr bevorzugtes Erscheinungsbild:

### Design wechseln
- Druecken Sie **Cmd+Umschalt+T** / **Strg+Umschalt+T**
- Oder verwenden Sie **Ansicht > Design umschalten** im Menue

### Design-Persistenz
Ihre Design-Praeferenz wird automatisch gespeichert und beim erneuten Oeffnen der Anwendung wiederhergestellt.

## Dokument-Metadaten {#metadata}

Legen Sie PDF-Metadaten beim Speichern Ihres Dokuments fest:

### Verfuegbare Felder
- **Titel:** Der Dokumenttitel
- **Autor:** Der Dokumentautor
- **Betreff:** Eine kurze Beschreibung des Inhalts
- **Schluesselwoerter:** Suchbare Schluesselwoerter fuer das Dokument

### So legen Sie Metadaten fest
1. Gehen Sie zu **Datei > Speichern unter...**
2. Ein Metadaten-Dialog erscheint vor dem Speicherdialog
3. Fuellen Sie die gewuenschten Felder aus (alle Felder sind optional)
4. Klicken Sie auf **Speichern**, um zum Dateispeicherdialog fortzufahren

## Automatische Updates {#auto-update}

Bleiben Sie mit den neuesten Funktionen und Fehlerbehebungen auf dem Laufenden:

### Funktionsweise
1. Beim Start prueft die App auf neue Versionen auf GitHub Releases
2. Wenn ein Update verfuegbar ist, sehen Sie eine Benachrichtigung
3. Klicken Sie, um das Update herunterzuladen und zu installieren
4. Die App startet mit der neuen Version neu

Derzeit erfolgen Update-Pruefungen automatisch beim Start. Zukuenftige Versionen koennten eine manuelle Pruefungsoption enthalten.

## Mehrfachauswahl-Operationen

Fuehren Sie Operationen an mehreren Seiten gleichzeitig durch:

### Seiten auswaehlen
- **Einzelne Seite:** Auf eine Miniaturansicht klicken
- **Bereichsauswahl:** Erste Seite anklicken, dann **Umschalt+Klick** auf letzte Seite
- **Auswahl umschalten:** **Cmd+Klick** / **Strg+Klick**, um Seiten zur Auswahl hinzuzufuegen oder zu entfernen
- **Alle auswaehlen:** Verwenden Sie **Cmd+A** / **Strg+A**

### Stapeloperationen
Sobald mehrere Seiten ausgewaehlt sind, koennen Sie:
- Alle ausgewaehlten Seiten auf einmal loeschen
- Alle ausgewaehlten Seiten in die gleiche Richtung drehen
- Alle ausgewaehlten Seiten zusammen verschieben (relative Reihenfolge beibehalten)

## Warnung bei ungespeicherten Aenderungen

Verlieren Sie nie Ihre Arbeit:

### Funktionsweise
- Der Fenstertitel zeigt "(Bearbeitet)" an, wenn Sie ungespeicherte Aenderungen haben
- Wenn Sie versuchen, das Fenster mit ungespeicherten Aenderungen zu schliessen, erscheint ein Bestaetigungsdialog
- Waehlen Sie, ob Sie speichern, Aenderungen verwerfen oder abbrechen und weiter bearbeiten moechten
